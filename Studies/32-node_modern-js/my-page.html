NodeJS & Modern JavaScript

NodeJS supports modern JavaScript features like ES Modules or Promises.

Imports with and without curly brackets {}

  • If we export using 'module.exports = something' you can import it directly. E.g: import something from 'somewhere'. Check 'responseHandlerWithExportDefault_WithPromises' function

  • Exporting the const directly like 'export const something = true' makes it necessary to add curly brackets on the import E.g: import { something } from 'somewhere'. Check 'responseHandlerExportedDirectly_WithCallback' function.

Using Global Variables

Global variables only work with commonJs, not with ES Modules (unless you create the __filename and __dirname consts making them available). Check 'responseUsingGlobalVariables' function

  • import.meta.url: globally available variable which gives you the path to this file name
  • fileURLToPath(): function that converts that URL to a path with which the path package then is able to work
  • dirname(): function that takes such a path to the current file to give you the path to the current folder in which this file sits
