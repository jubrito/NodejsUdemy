NodeJS & Modern JavaScript

NodeJS supports modern JavaScript features like ES Modules or Promises.

Imports with and without curly brackets {}

  • If we export using 'module.exports = something' you can import it directly. E.g: import something from 'somewhere'.

  • Exporting the const directly like 'export const something = true' makes it necessary to add curly brackets on the import E.g: import { something } from 'somewhere'
